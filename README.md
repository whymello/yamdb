# YaMDb API

REST API сервис публикации отзывов на произведения.

Проект реализуется как монолитное Django-приложение с использованием Django REST Framework.

## 📌 Текущий статус

Реализована основная функциональность API:

- регистрация пользователей
- подтверждение регистрации по email
- получение JWT-токена
- авторизация через JWT
- управление пользователями
- категории произведений
- жанры
- произведения
- отзывы на произведения
- комментарии к отзывам
- система ролей и прав доступа
- пагинация

Основная функциональность API реализована.

## 🚀 Реализованная функциональность

### 🔐 Аутентификация и регистрация (Auth)

- Регистрация: `POST /api/v1/auth/signup/`
- Отправка кода подтверждения на email
- Получение JWT-токена: `POST /api/v1/auth/token/`
- Аутентификация по JWT
- Подключён Simple JWT
- Доступ к защищённым ресурсам — только для аутентифицированных пользователей

### 👤 Пользователи (Users)

- Кастомная модель `User` расширена полями `bio` и `role`
- Реализованы:

  - управление пользователями (`/api/v1/users/`)
  - получение и редактирование своего профиля (`/api/v1/users/me/`)

- Администратор может:

  - создавать пользователей
  - редактировать информацию пользователей
  - удалять пользователей

### 🎬 Категории (типы) произведений (Categories)

- Эндпоинт: /api/v1/categories/
- Поддерживает:

  - получение списка категорий
  - создание категории
  - удаление категории

- Доступ:

  - чтение — всем пользователям
  - создание / удаление — только администраторам

### 🎬 Жанры (Genres)

- Эндпоинт: /api/v1/genres/
- Поддерживает:

  - получение списка жанров
  - создание жанра
  - удаление жанра

- Доступ:

  - чтение — всем пользователям
  - создание / удаление — только администраторам

### 🎬 Произведения (Titles)

- Эндпоинт: /api/v1/titles/
- Поддерживает:

  - просмотр произведений
  - создание произведений
  - редактирование
  - удаление

- У произведения есть:

  - название
  - год выпуска
  - описание
  - категория
  - жанры

- Также рассчитывается поле rating

Рейтинг рассчитывается автоматически как **средняя оценка отзывов пользователей**.

### ✍️ Отзывы (Reviews)

- Эндпоинт: /api/v1/titles/{title_id}/reviews/
- Поддерживает:

  - получение списка отзывов
  - создание отзыва
  - получение конкретного отзыва
  - редактирование отзыва
  - удаление отзыва

- Ограничения:

  - пользователь может оставить **только один отзыв** на произведение
  - оценка произведения должна находиться в диапазоне **1–10**

- Доступ:

  - чтение — для всех пользователей
  - создание — для аутентифицированных пользователей
  - изменение / удаление — для автора, модератора или администратора

### 💬 Комментарии к отзывам (Comments)

- Эндпоинт: /api/v1/titles/{title_id}/reviews/{review_id}/comments/
- Поддерживает:

  - получение списка комментариев
  - создание комментария
  - получение конкретного комментария
  - редактирование комментария
  - удаление комментария

- Доступ:

  - чтение — для всех пользователей
  - создание — для аутентифицированных пользователей
  - изменение / удаление — для автора, модератора или администратора

## 🚀 Установка и запуск

### Через uv (рекомендуется)

```bash
git clone https://github.com/whymello/yamdb_api.git
cd yamdb_api
uv sync
uv run manage.py migrate
uv run manage.py runserver
```

### Через pip

```bash
git clone https://github.com/whymello/yamdb_api.git
cd yamdb_api

python -m venv .venv
source .venv/bin/activate  # Linux/macOS
.\.venv\Scripts\activate   # Windows
pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
